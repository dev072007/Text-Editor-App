# Simple Code Editor (Python Tkinter)

A lightweight text editor built using Python and Tkinter. This application provides basic file management features such as creating, opening, editing, and saving text files through a simple graphical user interface.

## Features

* Create a new text file
* Open existing `.txt` files
* Edit text in a text area
* Save content to a text file
* User-friendly menu bar with file operations
* Success message after saving a file

## Technologies Used

* Python
* Tkinter (GUI Library)

## How It Works

The application uses Tkinter to create a graphical text editor window. Users can write or edit text, open existing text files, save their work, and create new files using the menu options. File operations are handled through Tkinter's built-in file dialog boxes.

## Learning Objectives

This project demonstrates:

* GUI development with Tkinter
* File handling in Python
* Menu bar creation
* Event-driven programming
* Text widget implementation
